# 🎮 Simple Game Tests (QA Automation Project)

## 📌 О проекте
Небольшой проект по автоматизированному тестированию игровой логики на C#.

Реализовано:
- тестирование бизнес-логики (монеты, покупки)
- проверка аналитических событий
- проверка логирования
- генерация отчётов Allure

---

## 🧪 Что тестируется

Игровая логика:
- добавление монет
- трата монет
- обработка недостатка ресурсов

Аналитика:
- события (purchase_success, purchase_failed)

Логирование:
- запись действий игрока

---

## ⚙️ Стек

- C#
- NUnit
- Allure Reports
- .NET
- GitHub Actions (CI)
